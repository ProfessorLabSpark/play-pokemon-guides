# Proportional Prize Models (Always Sum to 100%)

These models scale cleanly and fairly.

---

## Top 4 Model (4–8 players)

1st = 50%  
2nd = 25%  
3rd = 12.5%  
4th = 12.5%  

---

## Top 8 Model (9–32 players)

1st = 32%  
2nd = 20%  
3rd = 12%  
4th = 12%  
5th = 6%  
6th = 6%  
7th = 6%  
8th = 6%  

---

## Top 12 Model (24–32 players, optional)

1st = 20%  
2nd = 14%  
3rd = 10%  
4th = 10%  
5th = 6%  
6th = 6%  
7th = 4%  
8th = 4%  
9th = 3%  
10th = 3%  
11th = 2%  
12th = 2%
